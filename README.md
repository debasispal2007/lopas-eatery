# Lopa's Eatery 🍛
### A Heritage Atelier Experience for Bengali Cuisine

**Lopa's Eatery** is a premium, magazine-style mobile application designed to celebrate the rich tradition of Bengali flavors through a sophisticated, editorial lens. Moving away from generic food-ordering templates, this app focuses on a high-end "lifestyle magazine" aesthetic, featuring custom food photography and an elegant "Heritage Atelier" design system.

![Lopa's Eatery Preview](assets/images/cover.png)

## ✨ Key Features

*   **Editorial Menu:** A curated selection of authentic dishes like *Kosha Mangsho*, *Basanti Pulao*, and *Payesh*, each presented with high-quality photography and artisanal descriptions.
*   **Premium Design System:** Built on a custom palette of Terracotta (#B22222) and Warm Cream (#FEFCCF), utilizing sophisticated typography (Noto Serif and Manrope).
*   **Admin Approval Workflow:** A real-world business logic where users submit orders for review. The Admin (Owner) approves the order before payment is processed, ensuring ingredient availability and quality control.
*   **Mobile-First UX:** Smooth animations, glassmorphism elements, and a responsive layout built for the modern food lover.

## 🛠️ Tech Stack

*   **Framework:** [Flutter](https://flutter.dev/) (3.x)
*   **Language:** [Dart](https://dart.dev/)
*   **State Management:** [Provider](https://pub.dev/packages/provider)
*   **Typography:** [Google Fonts](https://pub.dev/packages/google_fonts) (Noto Serif, Manrope)

## 📱 Getting Started

### Prerequisites
*   Flutter SDK installed on your machine.
*   An Android or iOS Emulator, or a physical device.

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/lopas-eatery.git
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the app:
    ```bash
    flutter run
    ```

## 📦 Download the Preview
You can download the latest **APK** for Android devices from the [Releases](https://github.com/YOUR_USERNAME/lopas-eatery/releases) section. 

*Note: For the best experience, use the `app-arm64-v8a-release.apk` version.*

## 🎨 Design Philosophy
The app follows a "Heritage Atelier" philosophy—blending the rustic charm of traditional Bengali kitchens with the sleek minimalism of a high-end fashion atelier. Every interaction is designed to feel intentional, warm, and premium.
